## Install dendron-11ty

```bash
yarn init
yarn add @dendronhq/dendron-11ty
```

```
# start server
env LOG_LEVEL=info dendron-cli launchEngineServer --port 3010 --wsRoot .

# start eleventy
npx den-11ty --wsRoot=. --port 310 --serve
```
